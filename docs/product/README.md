# Meal Planner - Product Documentation

## Overview
An AI-powered meal planning application designed to reduce the time it takes to generate weekly meal plans while maintaining flexibility. Shared between 2 users (partners) with synchronized calendars and grocery lists.

## Primary Goal
Reduce the time it takes to generate a new meal plan for each week, while still allowing for flexibility in the meal planning process.

## Target Users
- 2 users (yourself and partner)
- Shared calendars and generated grocery runs

## Key Differentiator
**GenAI-Powered Planning** - The system uses AI to:
- Learn from historical meal plans and preferences
- Incorporate open-text rules and feedback
- Suggest meals based on patterns and constraints
- Adapt to user preferences over time
