# Phase 2 - Future Features

## Mobile Support
- Responsive web design for mobile browsers
- Consider Progressive Web App (PWA)
- Native mobile app (long-term consideration)

## Web Recipe Search
- Search the web for new recipes via AI
- Import recipes from various sources
- Save discovered recipes to local library

## Ingredient-Based Recipe Discovery
- "What can I make with what I have?" feature
- Input available ingredients
- AI suggests possible recipes

## Calorie Counting/Estimation
- Nutritional information tracking
- Calorie estimation for meals
- Daily/weekly nutritional summaries

## Servings Configuration
- Set number of people/servings per household
- Adjust ingredient quantities automatically
- Support variable serving sizes per meal

## Enhanced Calendar Integration
- Meal prep time blocking
- Reminder notifications
- Bidirectional sync with Google Calendar
- Shopping trip scheduling

## Recipe Scaling
- Automatically scale recipes up/down
- Smart unit conversions

## Inventory/Pantry Tracking
- Track what's in the pantry
- Suggest meals based on expiring ingredients
- Auto-update pantry after grocery runs
